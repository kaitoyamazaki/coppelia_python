# CoppeliaSim チュートリアル

本レポジトリでは, coppeliaSimの使用方法を覚えるために製作されたチュートリアルです.

本レポジトリの手順を踏むことで, 対抗二輪におけるライントレースシミュレーションを作成を目的としています.
